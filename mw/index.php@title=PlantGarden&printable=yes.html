Plant Garden

SPARQLthon85

Plantgarden植物ゲノムリソースのRDF化のために事前に設計済み。RDFポータル公開を見据えて川島さんからフィードバックをもらった

RDF設計

  • RDF summitとその後、SOのPropertyの代わりにRO、BFOのPropertyを利用したRDFを設計
  • Protein/PeptideとCDSの関係を記述するderives_fromは、obo:so_derives_fromではなくてobo:RO_0001000を利用
  • フィードバック反映
    • sio:has_annotation、prov:wasDerivedFromのリソース記載を除く
    • PlantGardenの実態のないURLについてブランクノードに変更

（参考）DDBJ RDFのSO→BFO プロパティ変更 

1.
<http://purl.obolibrary.org/obo/so_has_part>
 => <http://purl.obolibrary.org/obo/so#has_part>
 => <http://purl.obolibrary.org/obo/BFO_0000051>

2.
<http://purl.obolibrary.org/obo/so_part_of>
 => <http://purl.obolibrary.org/obo/so#so_part_of>
 => <http://purl.obolibrary.org/obo/BFO_0000050>

3.
<http://purl.obolibrary.org/obo/so_has_quality>
 => <http://purl.obolibrary.org/obo/so#has_quality>
 => <http://purl.obolibrary.org/obo/RO_0000086>
