第17回 SPARQLthon を下記日程で開催したいと思います。

開催概要

  • 開催期間：2014年2月17日(月) 10:00 〜 18日(火) 18:00
  • 開催場所：ライフサイエンス統合データベースセンター (DBCLS) 4階 415 (教室)
  • アクセス：http://dbcls.rois.ac.jp/access

プロジェクト

  • 次回以降の SPARQLthon 日程調整
    • SPARQLthon 用 Google カレンダー に学会等の予定を追記してください。書き込み権は片山・川島・岡本にメールでリクエストください。
    • スケジュールの空き状況を見て、候補日のリスト作り、SPARQLthon19 以降の日程を densuke などで希望調査してわりと先まで決めておこうと思っています。


  • TogoGenome/TogoStanza
    • UniProt 2014_01 + RefSeq 63 + EdgeStore (FALDO, クラス名, グラフ名改定対応)
    • ID mapping
    • NanoStanza
      • Environment: 生物種数, 生育温度分布, 生育pH分布, 写真, 系統プロファイル etc.
      • Organism: ゲノムサイズ, 遺伝子数, GC%, 生育温度, 生育pH, 表現型, application, disease, 系統プロファイル etc.
      • Gene: 遺伝子長, exon数, AS数, オーソログ/生物種数, 立体構造, 論文, EC, 立体構造 etc.
    • Human/Eukaryote genome RDF
    • Nanopub
    • 更新体制とサーバ移行準備
    • Stanzaサーバ公開手順
  • TogoDB/TogoWS
  • MicrobeDB
  • MBGD
  • DDBJ
  • GenomeRefine
  • RDF
    • VCF2RDF (BioInterchange/GVFO)
    • PubChem, EBI, PDBj
    • Glyco
    • BioResource
  • オントロジー
    • 菌株
    • taxdump 改定
    • PIERO
  • 横断検索サービスのメタデータの活用（MicrobeDB）
  • D2RQ : SPARQL -> SQL どうなるか検証
  • Neo4j : RDF -> Neo4j にロード (BioPAX データ)
  • 恒例のSPARQLエンドポイント付きトリプルストア調査
  • 新着論文レビューをMeSHから検索
  • グラフとネットワークの解析
  • メタボロのメタデータ
