開催概要

2019年7月8日から12日にかけて、指宿にて国内のライフサイエンスデータベースの構築者と利用者向けの技術勉強会として、第10回 国内版バイオハッカソン BH19.7 を開催いたします。

  • 主催：ライフサイエンス統合データベースセンター (DBCLS)
  • 共催：質量分析インフォマティクス研究会 (JSBi公募研究会)
  • 開催期間：2019年7月8日(月)〜7月12日(金)
  • 開催場所：休暇村指宿（鹿児島県指宿市）
    • コインランドリーあり（5台くらい）
    • ドローンによる休暇村〜知林ヶ島の空撮映像
  • アクセス（往路）
    • 空港バス乗り場
      • 指宿行き：１番のりば、鹿児島市内行き：２番のりば
      • 券売機、窓口は空港建物を出てすぐ、バス乗り場に面した通路にあります
        • ICカードは現地ICカードしか使えません
    • 鹿児島空港 -> 指宿駅
      • 連絡バス：95分, 2,350円, 時刻表
      • トイレ設備、トイレ休憩があるかは未確認（無さそう）
    • 鹿児島空港 -> 鹿児島中央駅
      • 連絡バス：40分, 1,250円, 時刻表
    • 鹿児島中央駅 -> 指宿駅
      • JR：75分 (特急55分), 1,000円 (特急1,000+1,140円),時刻表（喜入行き以外）
        • JR指宿駅は交通系ICカード未対応です
        • エリア外までICカードで乗車してしまった場合
      • 路線バス：98分, 950円, 時刻表（２ページ目右上）
        • 鹿児島中央駅 東16乗場
    • 鹿児島中央駅 -> 知覧（乗換）-> 指宿駅
      • 路線バス：遠回りになります。詳細はサイト参照
        • 知覧から指宿駅へのバスは本数が少ないのでご注意下さい
    • 指宿駅 -> 休暇村 指宿
      • 送迎バスの乗り場は、JR駅入口（東側）とは逆側になります。駅を出て左手すぐの地下連絡通路を通って駅の西側に回る必要があります。路線バス、空港連絡バスは駅前（東側）左手の停留所になります。（map）
      • 送迎バス：10分, 無料, 時刻表
        • 指宿駅発 15:10, 16:25, 17:25
      • 臨時送迎バス：10分, 無料
        • 往路 7/7 13:10, 14:10 乗車場所はJR指宿駅西側（空港バスが遅れた場合最大10分待機しますが、乗れなかった場合は別の手段をお使い下さい）
        • 復路 7/12 時刻詳細は後日掲載
      • 路線バス：10分, 150円, 時刻表（１ページ目下の右の方）
        • 現金のみ、高額紙幣両替不可
      • タクシー：10分, 1,000円程度
      • 徒歩：40分
  • 復路メモ
    • 指宿駅前発空港連絡バスは前払い、高額紙幣両替不可

開催趣旨

DBCLS では、ライフサイエンスのデータベース統合についての技術開発を支援・最新技術の情報共有・実務者間での交流を促進するため、これまで11回の国際版 BioHackathon と9回の国内版バイオハッカソン BH10.10, BH11.11, BH12.12, BH13.13, BH14.14, BH15.15, BH16.12, BH17.11, BH18.7を開催してまいりました。

バイオインフォマティクスにおけるデータベースの利活用は目的も手法も多岐にわたります。このため、国内版バイオハッカソンのテーマについては DBCLS や NBDC の統合化推進プログラムで進められているセマンティック・ウェブによるデータベース統合に限らず、国内の生命情報学系の研究者間で共にディスカッションやソフトウェア開発が促進できるものであれば全てスコープに入ると考えます。また、民間の方からもぜひ、公共データの社内での活用や情報統合の課題、産学連携で開発・利用できそうな研究課題やソフトウェア技術など、バイオ寄りからインフォ寄りまで議論・開発のネタを自由に持ち込んで頂ければと思います。

国内版バイオハッカソンは、これからのライフサイエンスを支える技術開発に興味のある方々に自由にご参加いただき、活発な議論と開発を進めて頂くインキュベーションの場となればと考えています。
