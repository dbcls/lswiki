d3.js Data-Driven Documents

  • サイト：http://mbostock.github.com/d3/
  • 技術：JavaScript, SVG

bio-js

  • サイト：http://code.google.com/p/bio-js/
  • 技術：JavaScript, prototype.js

Spark

  • サイト：http://km.aifb.kit.edu/sites/spark/
  • 技術：JavaScript, SPARQL

ゲノム座標情報

Dalliance Genome Explorer

  • サイト：http://www.biodalliance.org/
  • デモ：http://www.biodalliance.org/human/ncbi36/ -- NCBI36
  • 技術：JavaScript, SVG, DAS

jBrowse

  • サイト：http://gmod.org/wiki/JBrowse
  • デモ：http://jbrowse.org/genomes/dmel/
  • 技術：JavaScript, HTML5, JSON

RelFinder

  • サイト：http://relfinder.dbpedia.org/
